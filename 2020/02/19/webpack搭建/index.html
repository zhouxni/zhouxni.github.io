webpack准备

在项目中安装webpack，webpack-cli，webpack-dev-server三个基本运行包。同时全局安装webpack和webpack-cli。

创建配置文件

在根目录下创建webpack.config.js文件。

webpack常见配置项

  1. entry入口文件

  2. output输出文件

mode:”production”//代表生产环境
mode:”development”//代表开发环境

devServer:{//开发环境配置
contentBase:path.resolve(__dirname,”dist”),//打包目录
port:8100,//开启端口
compress:true,//对所有服务开启gzip压缩
hot:true,//启用热模块更换
open:true,//运行后，浏览器自动打开
overlay:true//警告和错误是否在浏览器屏幕上显示
overlay:{
errors:true,
warnings:true
}
},
```
其它常见配置，可以参看webpack;
