学习uni-app--vue注意事项

2019-07-26
  • 前端

介绍

uni-app 在发布到H5时支持所有vue的语法；发布到App，由于平台限制，无法实现全部vue语法，但uni-app仍是是对vue语法支持度最高的跨端框架。本文只讲解有差异的地方

Class 与 Style 绑定

非H5端不支持 Vue官方文档：Class 与 Style 绑定 中的 classObject 和 styleObject 语法。

表单控件绑定

虽然说支持vue的官方表单控件绑定v-model，但是建议用：value方式进行绑定，与小程序设定保持一定。
H5 的select 标签用 picker 组件进行代替。
表单元素 radio 用 radio-group 组件进行代替

v-html指令

在App端是没有DOM概念的，因此v-html无效。不过可以用rich-text富文本组件进行代替。

v-show指令

v-show在App端某些场景表现异常，建议使用v-if代替。

组件

非H5端不支持列表：

  • 动态组件
  • 异步组件
  • inline-template
  • keep-alive
  • Slot中scoped 暂时还没做支持
  • X-Templates
  • transition(可用animation和css动画代替)

全局数据和方法

vuex

uniapp中内置vuex，用法与H5一样，唯一不同的是将store进行挂载时，要将store同时挂载到vue.prototype上(多出的一步操作)，通过原型属性去调用。

globalData

在App.vue中声明globalData对象，这是小程序的技术机制。在各个页面通过getApp().globalData就可获取数据。这种方式一般用于vue页面和nvue页面进行数据共享和传递。
