webpack打包启用gzip

webpack在打包时可以借助 compression webpack plugin 实现gzip压缩，首先需要安装该插件。在vue.config.js配置如下:


生成的压缩文件以.gz为后缀
一般浏览器都已支持.gz的资源文件，在http请求的Request Headers 中能看到 Accept-Encoding:gzip。

服务器配置

开启 gzip on;gzip_static on;。其中gzip_static on最为关键，它表示nginx会优先匹配你的gzip文件来返回，如果没有就寻找相应资源进行gzip压缩再返回(前提是服务器已进行配置额外压缩操作，否则以前端gzip压缩为准，没压缩的文件就返回源文件)。
