vue-路由传参

2019-08-27
  • 前端

介绍

路由传参通常用两种方法：params和query

  • params：
    如：/user/id=>/user/:id
  • query：
    如：/user?id=2=>/user

props

在组件中使用 $route 会使之与其对应路由形成高度耦合，从而使组件只能在某些特定的 URL 上使用，限制了其灵活性。

  • $route
  • props 当设置props为true时，$route.params将会被设置为组件属性，即$route.params.id将会被映射为组件props中的id属性。
    如果 props 是一个对象，它会被按原样设置为组件属性。当 props 是静态的时候有用。
    设置props为函数

注意

使用编程式导航时，使用path，不能带params对象形式传参。
